## タグ付けルール

収集した投稿に以下のタグを付け、`docs/research/listening-log.md` に記録する:

| タグ | 意味 |
|---|---|
| `#終演後` | 終電・規制退場・夜の移動に関する言及 |
| `#高騰満室` | 価格高騰・満室・取れない |
| `#直前` | チケット当選後の直前予約の苦労 |
| `#安全` | 女性ひとり・治安・夜道 |
| `#同行` | 連番相手・割り勘・相部屋 |
| `#手段` | 使っているアプリ・サイト・探し方 |

## 記録フォーマット

```
日付 | タグ | 要旨（引用は避けて要約） | 会場名（あれば）
```

※ 投稿の全文転載はしない（プライバシー・利用規約への配慮）。要約と件数だけを残す。
