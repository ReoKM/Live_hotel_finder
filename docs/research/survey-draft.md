# 定量アンケート票（GATE 1 通過後に配布）

- 目的: インタビューで生き残ったペインの優先順位を定量化する
- 目標回答数: n=50以上
- 配布: X・推し活コミュニティ・インタビュー協力者経由
- 所要: 3分以内（離脱防止のため12問以内厳守）

---

## スクリーニング

**Q1. 過去1年以内に、宿泊を伴うライブ・イベント遠征をしましたか？**
- はい（→続行）
- いいえ（→終了）

**Q2. 過去1年の宿泊遠征の回数は？**
- 1回 / 2〜3回 / 4〜6回 / 7回以上

## 行動実態

**Q3. 前回の遠征で、宿探しに使ったものをすべて選んでください**
- 楽天トラベル / じゃらん / Booking.com / agoda / トリバゴ等の比較サイト / Googleマップ / X(Twitter)の口コミ / 公式ツアー・ホテルセット / その他（自由記述）

**Q4. 宿を決めるまでにかかった時間は？**
- 10分以内 / 30分以内 / 1時間以内 / 数日かけて悩んだ

**Q5. 宿を予約したタイミングは？**
- チケット応募と同時（当選前） / 当選直後 / 公演1ヶ月前 / 1週間前〜直前

## ペインの優先順位（核心）

**Q6. 遠征の宿探しで困った経験をすべて選んでください**
- 会場周辺が満室で取れなかった
- 価格が高騰していた
- 終演後に帰れるか（終電・移動時間）の確認が面倒だった
- 土地勘がなくどのエリアが良いか分からなかった
- 複数サイトの価格比較が面倒だった
- 夜遅くの一人歩きが不安なエリアだった
- 同行者との調整が面倒だった
- 困った経験はない

**Q7. Q6で選んだ中で、一番のストレスは？（1つ）**

**Q8. 宿を決めるとき、最も重視するものは？（1つ）**
- 価格 / 会場からの近さ・移動時間 / 駅からの近さ / 部屋の質・広さ / 評価・口コミ / チェックイン/アウト時刻

## コンセプト受容性（最後に置く）

**Q9. 「会場を選ぶと、終演後に帰りやすい宿が移動時間つきで一覧できるサイト」があったら、次の遠征で使ってみたいですか？**
- ぜひ使いたい / 使ってみてもいい / あまり使わない / 使わない

**Q10. （Q9で「使いたい」選択者）そのサイトに一番期待することを一言で**（自由記述）

**Q11. 年代**: 〜19 / 20代 / 30代 / 40代 / 50代〜

**Q12. 差し支えなければ、主な遠征ジャンル**（自由記述・任意）

## 集計の判定基準（GATE 2）

- Q6で「終演後に帰れるかの確認が面倒」が**40%以上** → 終演後シミュレーターを最優先で開発
- Q8で「会場からの近さ・移動時間」が価格を上回る → 意思決定支援路線を確定
- Q9の「ぜひ使いたい」が**30%未満** → コンセプトの伝え方 or 対象セグメントを再検討
